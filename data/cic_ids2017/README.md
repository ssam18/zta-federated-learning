# CIC-IDS2017 Dataset

## Overview

This directory contains network intrusion data collected at the Canadian Institute for
Cybersecurity (CIC) using a purpose-built testbed that mirrors a realistic enterprise network
topology. Traffic was recorded over five days (Monday through Friday, July 3–7, 2017) on a
controlled network segment hosting web servers, file servers, and end-user workstations.

## Collection Environment

- **Testbed**: CIC network lab with 50 machines (Windows, Ubuntu, macOS)
- **Capture points**: Network taps on each VLAN segment
- **Tool**: CICFlowMeter for bidirectional flow feature extraction
- **Benign traffic**: Generated by B-Profile, a behaviour-based profile of real user activity
- **Attack infrastructure**: Dedicated Kali Linux attacker nodes per attack category

## Traffic Categories

| Category | Days Captured | Description |
|----------|--------------|-------------|
| BENIGN | All days | Normal enterprise user activity |
| DoS Hulk | Wednesday | HTTP flood using Hulk tool |
| DDoS | Friday | LOIT-based distributed denial-of-service |
| PortScan | Friday | nmap SYN scan across full port range |
| Bot | Thursday | Botnet command-and-control and propagation |
| Web Attacks | Thursday | Brute force, XSS, SQL injection |
| Infiltration | Thursday | Dropbox upload and Portscan via Metasploit |

## Files

| File | Description |
|------|-------------|
| `raw/network_flows.csv` | Bidirectional flow records with 78 features per flow |
| `processed/` | Preprocessed versions with normalised features and encoded labels |

## Feature Set

Each row represents one bidirectional network flow. The 78 features cover:
- Basic flow statistics (duration, packet counts, byte counts)
- Packet length statistics (min, max, mean, std)
- Inter-arrival time statistics for forward and backward directions
- TCP flag counts (FIN, SYN, RST, PSH, ACK, URG)
- Active and idle time statistics
- Subflow statistics and window sizes
